# travel-ai-test

## Setup

### Frontend
- `cd /home/ubuntu/Code/travel-ai-test/travel-ai-app`
- `npm install`
- `npm run dev`

### Backend
- `cd /home/ubuntu/Code/travel-ai-test/server`
- `npm install`
- Set environment variables:
  - `OPENAI_API_KEY`
  - `AMADEUS_CLIENT_ID`
  - `AMADEUS_CLIENT_SECRET`
  - `PORT` (optional, defaults to 5179)
- `npm run dev`
